# 🕊️ Site Jovens Efatá

Landing page para o **Jovens missionários Efatá**, um grupo que busca promover missão, espiritualidade e relacionamento
em comunidade dentro da igreja católica romana e na sociedade.

------------------------------------------------------------------------

## 📌 Sobre

Esse projeto é a página oficial do **Grupo de Jovens Efatá** --- um
espaço feito para divulgar atividades, eventos e fortalecer a identidade
da juventude cristã. O site serve como ponto de contato e apresentação
da missão, visão e valores do grupo.

------------------------------------------------------------------------

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido com as seguintes tecnologias:

-   TypeScript\
-   Vite (ferramenta de build)\
-   HTML & CSS\
-   Outras dependências definidas no `package.json`

## 🛠️ Como Rodar Localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

1.  Clone o repositório:

    ``` bash
    git clone https://github.com/NicolasRSJ/site-jovens-efata.git
    ```

2.  Entre na pasta do projeto:

    ``` bash
    cd site-jovens-efata
    ```

3.  Instale as dependências:

    ``` bash
    npm install
    ```

4.  Crie seu arquivo de ambiente a partir do exemplo:

    ``` bash
    cp .env.example .env
    ```

5.  Inicie o servidor de desenvolvimento:

    ``` bash
    npm run dev
    ```

6.  Abra o navegador no endereço que aparecer no terminal (ex:
    `http://localhost:5173`).

------------------------------------------------------------------------

## 📦 Como Buildar para Produção

Para gerar a versão de produção do site, basta rodar:

``` bash
npm run build
```

Os arquivos finais serão gerados na pasta `dist`.
